Step 1. 메인시트에서 키워드 확인

1메인시트 열기 (보기 전용)
메인시트에서 작업할 키워드를 찾습니다.
카페 링크(A열)가 비어있는 키워드가 작업 대상입니다.

메인시트 예시

A (카페 링크)C (키워드)D (월검색량)F (키워드분류)G (카테고리)H (인기글순서)K (카)
2 — 비어있음 — 안경점 66,020 일반 안경 스마트블럭 못잡음
3 https://cafe.naver.com/... 루테인 49,350 필수 성분 신뢰도통합 상단
4 — 비어있음 — 녹내장 45,340 일반 녹내장 S블록
5 승주 루테인지아잔틴 41,310 필수 성분 스마트블럭
💡 확인 포인트
• A열이 비어있으면 → 작업 가능 ✅
• A열에 이름이 있으면 → 다른 사람이 찜한 거 ❌
• A열에 링크가 있으면 → 이미 완료된 거 ❌
• 카테고리와 인기글 순서를 보고 글 작성 방향을 정합니다.

Step 2. 내 작업시트에 키워드 입력 (찜)

2B열에 키워드 입력
내 작업시트의 B열(키워드)에 작업할 키워드를 입력합니다.
입력하는 순간 자동으로 메인시트에 내 이름이 들어갑니다. (찜 완료)

내 작업시트 — 키워드 입력 후

A (날짜)B (키워드)C (카페링크)D (상태)E (상위노출현황)
2 2026-03-23 안경점 📌 찜 완료 못잡음
3 2026-03-23 녹내장 📌 찜 완료 -
✅ 자동으로 일어나는 일
• D열: "📌 찜 완료" 자동 표시
• E열: 메인시트의 상위 노출 현황 자동 표시
• A열: 날짜 자동 입력
• 메인시트 A열: 내 이름 자동 입력 (다른 사람이 못 가져감)

Step 3. 카페에 게시글 작성

3네이버 카페에 글 작성
메인시트에서 확인한 카테고리와 인기글 순서를 참고하여
해당 키워드에 맞는 카페 게시글을 작성합니다.
작성 후 게시글 URL을 복사합니다.

Step 4. 카페링크 입력 → 자동 반영

4C열에 카페링크 붙여넣기
내 작업시트의 C열(카페링크)에 복사한 URL을 붙여넣습니다.
붙여넣는 순간 자동으로 메인시트에 반영됩니다.

내 작업시트 — 링크 입력 후

A (날짜)B (키워드)C (카페링크)D (상태)E (상위노출현황)
2 2026-03-23 안경점 https://cafe.naver.com/test/12345 ✅ 완료 다음 체크 후 반영
3 2026-03-23 녹내장 https://cafe.naver.com/test/67890 ✅ 완료 다음 체크 후 반영
✅ 자동으로 일어나는 일
• D열: "✅ 완료"로 변경
• 메인시트 A열: 내 이름 → 카페링크로 교체
• 상위노출현황: 다음 새벽 순위 체크 후 자동 반영

⚠️ 이런 경우가 있어요

⚠️다른 사람이 이미 찜한 키워드
D열에 "❌ OO님이 작업 중"이 뜹니다. 다른 키워드를 선택하세요.
⚠️이미 링크가 있는 키워드
D열에 "❌ 이미 완료된 키워드"가 뜹니다. 다른 키워드를 선택하세요.
⚠️메인시트에 없는 키워드
D열에 "❌ 메인시트에 없는 키워드"가 뜹니다. 키워드를 정확하게 입력했는지 확인하세요.

📊 상위 노출 현황 보는 법

E열에 표시되는 상위 노출 현황은 매일 새벽 자동으로 업데이트됩니다.
내가 작성한 카페글이 네이버 검색에서 어디에 위치하는지 보여줍니다.

다음 체크 후 반영오늘 작성한 글, 내일 확인됨-

✅ 지켜야 할 것 / ❌ 하면 안 되는 것

✅ DO

  • 내 작업시트에서만 작업하기
  • 키워드를 정확하게 입력하기
  • 카페링크는 전체 URL 붙여넣기
  • 메인시트에서 카테고리/인기글순서 확인하기
  • 작업 전 A열 확인 (빈 것만 작업)

❌ DON'T

  • 메인시트 직접 편집 ← 절대 금지!
  • 다른 사람이 찜한 키워드 작업하기
  • 같은 키워드 중복 작업하기
  • 카페링크 대신 다른 내용 넣기
  • 메인시트 행/열 드래그하기

🔍 메인시트에서 필터 사용하는 법

메인시트는 보기 전용이지만, 필터 뷰를 사용하면 원하는 키워드만 골라서 볼 수 있습니다.
다른 사람에게 영향 없이 나만의 필터를 걸 수 있습니다.
1필터 뷰 열기
상단 메뉴에서 데이터 → 필터 보기 만들기를 클릭합니다.

📌 메뉴 위치
데이터 → 필터 보기 만들기
또는 상단 필터 아이콘(깔때기 모양) 옆의 ▼ 화살표를 클릭해도 됩니다.
2원하는 조건으로 필터 걸기
필터 뷰가 활성화되면 각 열의 헤더에 ▼ 아이콘이 나타납니다.

예시: 카테고리별로 보기
① 카테고리 열의 ▼ 클릭
② "값으로 필터링"에서 원하는 카테고리 선택
    예: "성분"만 체크 → 성분 관련 키워드만 보임
③ 확인 클릭
예시: 작업 안 된 키워드만 보기
① 카페 링크(A열)의 ▼ 클릭
② "값으로 필터링"에서 "(빈칸)"만 체크
③ 확인 클릭
→ 카페 링크가 비어있는(= 작업 안 된) 키워드만 보입니다!
예시: 특정 인기글 순서만 보기
① 인기글 순서 열의 ▼ 클릭
② "신뢰도통합", "인기글" 등 원하는 값만 체크
③ 확인 클릭
3필터 뷰 닫기
작업할 키워드를 확인했으면, 오른쪽 위의 ✕ 버튼을 눌러 필터 뷰를 닫습니다.
닫아도 원본 데이터에는 영향 없습니다.
💡 필터 뷰 vs 일반 필터
• 필터 뷰: 나만 보임, 다른 사람에게 영향 없음 ✅
• 일반 필터: 모든 사람에게 적용됨 ❌ (사용 금지!)

반드시 "필터 보기 만들기"를 사용하세요. 일반 필터를 걸면 다른 사람 화면도 바뀝니다.
⚠️주의
필터는 보기만 바꾸는 것입니다. 데이터를 삭제하거나 수정하는 게 아닙니다.
필터를 걸어도 메인시트 원본은 그대로입니다.

❓ 자주 묻는 질문

Q. 찜한 키워드를 취소하고 싶어요
작업시트에서 해당 행의 키워드를 지우면 됩니다. 메인시트의 이름도 자동으로 사라집니다.
Q. 상위노출현황이 "다음 체크 후 반영"이에요
오늘 작성한 글은 내일 새벽 순위 체크 후 결과가 나옵니다. 하루 기다려주세요.
Q. 메인시트에서 필터를 걸고 싶어요
데이터 → 필터 뷰 → 임시 필터 뷰 만들기를 사용하세요. 다른 사람에게 영향 없이 필터를 걸 수 있습니다.
🚨 중요!
문제가 생기면 메인시트를 절대 건드리지 말고 담당자에게 연락하세요.
